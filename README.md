CinePass - Ticket movie booking platform

Techstack - React.js, Tailwind CSS, react-router-dom for navigation through the website, Lucide-react for icons, react-hot-toast for receiving notifications, react-player for playing videos, mongodb atlas for creating and maintaining database.

Clerk for authentication, inngest is used for performing background jobs

prebuiltui.com - used for using online available data components for footer

TMDB (The Movie Database) API - used for fetching real-time movie data

Inngest - Inngest is a developer platform for background jobs and event-driven workflows. It lets you easily run async, delayed, or scheduled functions in your app — without managing cron jobs, queues, or separate worker servers.

Payment integration - Stripe payment Sample card details - Card Number - 4242 4242 4242 4242  Expiry date - any future date CVV - any 3-digit number Name on card - any name of your choice
